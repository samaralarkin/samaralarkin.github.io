Independent Game Development

Atom Dancer

squidlarkin.itch.io/atom-dancer

My first commercial game project. I had created a simple simulation of objects gravitationally attracting each other, and was intrigued by how they clustered together. I came up with the idea of creating a non-violent game about interacting with these chaotic systems. The "bond-crossing" mechanic, introduced to increase both risk and reward, turned out to be the most technically challenging part of the gameplay to implement.

I created the game in Game Maker 8, using around 250 lines of GML code. The project taught me a lot about polishing and shipping a project, including such details as supporting multiple screen resolutions, graphic design, and documentation.

FormVaders

http://www.glorioustrainwrecks.com/node/3386

A joke game, programmed in under two hours in C# using Visual Studio. The goal was to use Windows Forms, particularly the DataGridView element, in ways they were never meant to be used. I succeeded.

@spellwithout

http://twitter.com/spellwithout

A Twitter bot that finds words inside other words. Written in Python, and based on an existing framework which I downloaded from Github and added about 60 lines of code to. Challenges included detecting and discarding uninteresting results, as well as logging to prevent repeated tweets.
